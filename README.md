# Test Alignet Wallet

Provided input data is located on `.env` file.

To setup:
```
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```

To test wallet SOAP service:
```
python wallet.py
```

To transactions query:
```
python transactions.py
```

To web app to send data to VPOS2:
```
python web.py
```
